NTM Restricted Registration for WP-Members
==========================================

Author: New Tribes Mission (Stephen Narwold)
WordPress Plugin for use in conjunction with WP-Members. Assumes WP native registration is turned off.
Restricts registration to email addresses listed within this file (edit the file to add/remove/edit email addresses or domains). Includes both whitelist (accepted emails) and blacklist (blocked emails). The blacklist will override entries in the whitelist
